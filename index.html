Project details

HTML/CSS

HTML: Used to structure and organize the content on the webpage, such as the map area, buttons, and filter options.

CSS: Applied to style the webpage, controlling the visual presentation of the HTML elements, including colors, fonts, and layout to achieve a pleasing aesthetic and user-friendly interface.

Javascript ES6

Javascript: is the driving force behind the site's interactivity, handling user inputs, filter toggles, map updates, and integration with the Google Maps API to dynamically display locations based on user preferences.

Bootstrap 5

Bootstrap is used for its responsive design features, ensuring the website is adaptable to various screen sizes and devices. It provides a grid system, responsive utilities, and pre-styled components for quick and efficient UI development.

Google Maps API

Google Maps API: is integrated to render the interactive map, providing geographic data, navigation, and location markers. It allows users to visually explore the area and find places of interest such as skating rinks, museums, and waterfalls.
